# She Can Foundation Website

A simple and responsive single-page NGO website created using HTML, CSS, and JavaScript.

This project was developed as a Frontend Development Internship Task for She Can Foundation.

---

## 🌟 Features

- Responsive Navigation Bar
- Hero Section
- About Section
- Services Section
- Contact Form
- Compact Footer
- Mobile Responsive Design
- Interactive Hamburger Menu
- Hover Effects & Animations
- Clickable Email Address
- Social Media Icons
